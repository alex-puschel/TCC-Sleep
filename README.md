# Sleep Analysis using Linear Regression

This project analyzes the CMU Sleep dataset using multivariable linear regression, vectorization, and gradient descent.

## Objective

Evaluate how academic, demographic, and sleep-related variables influence Total Sleep Time.

## Dataset

CMU Sleep dataset from Kaggle.

## Methods

- Exploratory Data Analysis
- Feature normalization
- Multivariable Linear Regression
- Gradient Descent
- Model evaluation using MAE, RMSE and R²
